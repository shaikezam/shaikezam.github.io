add(int index, Element e)
 Add an element in the index position, returns void 
get(int index)
 Get an element in the index position 
remove(int index)
 Remove an element in the index position and returned the removed element 
set(int index, Element e)
 Replace an element in the index position and returned the old element 
addAll(int index, Collection c)
 Add elements from the index position 
indexOf(Object o)
 Get the first occurrence of an object in the list, otherwise returns -1 
lastIndexOf(Object o)
 Get the last occurrence of an object in the list, otherwise returns -1 
subList(int fromIndex, int toIndex)
 Return sub list from the given indices
