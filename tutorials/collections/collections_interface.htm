size()
 Returns the number of elements in this collection 
isEmpty()
 Returns true if this collection contains no elements 
contains(Object)
 Returns true if this collection contains the specified element 
iterator()
 Returns an iterator over the elements in this collection. 
add(E)
 Returns true if this collection changed as a result of the call or false if this collection does not permit duplicates and already contains the specified element. 
remove(Object)
 Removes a single instance of the specified element from this collection, if it is present 
containsAll(Collection)
 Returns true if this collection contains all of the elements in the specified collection 
addAll(Collection)
 Adds all of the elements in the specified collection to this collection 
removeAll(Collection)
 Removes all of this collection's elements in the specified collection 
retainAll(Collection)
 Retains only the elements in this collection that are contained in the specified collection 
clear()
 Removes all of the elements from this collection, the collection will be empty after this method returns.
